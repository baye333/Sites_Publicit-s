# Mon Portfolio Web

Bienvenue sur mon site web personnel ! Ce projet présente mon portfolio, mes compétences et mes projets.

## Structure du projet

- `index.html` : Page d'accueil
- `portfolio.html` : Page portfolio
- `a-propos.html` : Page "À propos"
- `style/` : Dossier contenant les fichiers CSS pour le style
- `images/` : Dossier contenant les images (accueil et portfolio)

## Technologies utilisées

- HTML5
- CSS3

## Comment utiliser

1. Ouvrez `index.html` dans votre navigateur pour accéder au site.
2. Naviguez entre les pages via les liens.
